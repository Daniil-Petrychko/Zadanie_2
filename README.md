Aplikácia s grafickým rozhraním pre triedenie 14 kódov miestností BA1 a S31. 
GUI obsahuje 2 listboxy pre oddelené zobrazenie blokov BA1 a S31, vstupné textové pole s placeholder textom, panel štatistík a tlačidlá pre fixnú sadu, náhodnú sadu, spracovanie, vyčistenie a kopírovanie výsledku. 
Funkcie: automatické triedenie podľa čísla, detekcia duplicít so zvýraznením, validácia neplatných kódov, štatistiky s percentuálnym rozdelením.
